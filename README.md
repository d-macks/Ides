## Usage

**Prefix**
> ['**s!**']

### **schedule**
**Aliases**  
> ['**s**']

**Argument 1**
> Division (**Mandatory Argument**)

**Argument 2**
> Events shown (Optional Argument, if not supplied, all the events will be shown)

**Usage**
> s!s PBST 5  
> *Shows **5** events from the security schedule*

### nextEvent
**Aliases**
> [**ne**]

**Argument 1**
> Division (**Mandatory Argument**)
 
**Usage**
> s!ne PBST  
> *Shows **the next** event from the security schedule*
